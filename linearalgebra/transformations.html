definition

A linear transformation is a mapping between vector spaces.

We represent mapping between two vector spaces with an arrow indicating which way we are mapping. So if we map from `RR` to `RR^2`, we indicate this by `RR rarr RR^2`.
Linear transformations must have the two following properties:

For a linear transformation `L: RR^n rarr RR^m`
  1. For vectors `x,y in RR^n`
    `L(x + y) = L(x) + L(y)`
  2. For a scalar `alpha in RR`
    `L(alpha x) = alpha L(x)`

some examples

If you're like me, definitions are great but you need to become familiar with them in order to understand, so let's go through some examples of linear transformations.

  1. very simple

    A very simple linear transformation between `RR^2` and `RR^2` is
    `L_1(x_1,x_2)=(x_1, x_2)`
    All this transformation does is take a vector from `RR^2` and maps it back to the exact same vector!
    I guess this example isn't much of a transformation, but we should ensure it meets the two definition properties:
    1. To ensure property 1 holds for the map `L_1` we need two arbitrary vectors from `RR^2`. Let's call them `x=(x_1, x_2)` and `y=(y_1, y_2)`.
      $$\begin{align} L_1(x+y)&=L_1(x_1+y_1, x_2+y_2)\\ &=(x_1+y_1, x_2+y_2) \end{align}$$ and $$\begin{align} L_1(x) + L_1(y)&=L_1(x_1, x_2) + L_1(y_1, y_2)\\ &=(x_1, x_2)+(y_1,y_2)\\ &=(x_1+y_1, x_2+y_2) \end{align}$$ And here we see that `L_1(x+y)=L_1(x)+L_1(y)` and so property 1 does hold.
    2. To check property 2 we have to use an arbitrary scalar, let's call it `alpha in RR` and an arbitrary vector from `RR^2`, let's call it `x=(x_1, x_2)`. $$\begin{align} L_1(\alpha x) &=L_1(\alpha x_1, \alpha x_2)\\ &=(\alpha x_1, \alpha x_2)\\ \end{align}$$ and $$\begin{align} \alpha L_1(x) &=\alpha L_1(x_1, x_2)\\ &=\alpha (x_1, x_2)\\ &=(\alpha x_1, \alpha x_2)\\ \end{align}$$ So we see that property 2 is satisfied as `L_1(alpha x) = alpha L_1(x)`.

      So while the linear transformation we defined doesn't really transform anything it certainly is a linear transformation.

    Whew, I know I called this example 'very simple' but we did cover quite a bit. We saw our first linear transformation and proved that it was in fact a linear transformation. This kind of mapping is called the identity transformation of `RR^2.`
